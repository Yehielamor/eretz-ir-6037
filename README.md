# eretz-ir-6037

A real-time, multiplayer Hebrew version of the classic word game 'Country, City, Thing' (Eretz, Ir, Davar in Hebrew), where players compete in timed rounds to list Hebrew words for categories starting with a given letter. The game features live scoring, chat, and leaderboards to enhance social interaction and language learning.

## Tech Stack
Vanilla HTML/CSS/JS + Supabase

## Setup
Run `docker-compose up` to start the infrastructure.
